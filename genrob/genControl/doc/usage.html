Benutzung

 genRob®-genControl ist eine kommandozeilenorientierte Anwendung ohne grafische Oberfläche.

In diesem Kapitel finden Sie Details zur Funktionsweise, um ein Gefühl für den Server zu bekommen.  Hier wird auch beschrieben, was für Voraussetzungen zu erfüllen sind, damit ein Server zum Laufen gebracht werden kann.

Die Ausführung mit möglichen bzw. teilweise notwendigen Parametern wird erläutert und mögliche Konsole-Befehle beschrieben.
