Voraussetzungen

Java™

 Das Produkt wurde unter "Java™ 2 Platform - Standard Edition" (wie von der Sun Microsystems, Inc., USA spezifiziert) in der Version 1.4.2 getestet.  Eine aktuelle Version kann für verschiedene Betriebssysteme von Sun's Java™-Website geholt werden.  Für Computer der Firma Apple sind nähere Informationen dazu hier zu finden - neuere Geräte (ab ca. Mitte 2004) sind bereits mit einer passenden Java™-Version ausgestattet.

Die in dieser Dokumentation erwähnten Batch- bzw. Script-Dateien erwarten, daß sich das bin-Verzeichnis der Java™-Installation im Suchpfad für Programme befindet.

Alle neueren Rechner jünger als 4 Jahre sollten genügen.

Einrichtung

genRob®-genControl ist bis auf eine möglicherweise notwendige Parametrisierung sofort einsatzbereit.  Der Server kann allein oder mit passenden Modulen laufen.  Zweckmäßig ist die Erstellung einer passenden Batch/Script-Datei, anstatt bei jedem Start alle Parameter einzugeben.  Details unter Ausführung.

Zu beachten ist, daß für einen produktiven Einsatz die originale Bibliothek des Herstellers von Jini™ verwendet werden muß.
