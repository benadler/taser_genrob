Überblick

 Eine Sitzung ist ein Modus von genRob®-genView, in dem Meßwerte wie z.B. Temperatur und Feuchte erfaßt, angezeigt, abgelegt und später jederzeit nachbearbeitet werden können.

Für eine Sitzung müssen verschiedene Bedingungen erfüllt sein:

  • Beim Start von genView müssen auf der Kommandozeile die Java™-Eigenschaften genRob.genView.genPCAN.properties und genRob.genView.genPCAN passend mit dem Namen der CAN-Konfigurationsdatei bzw. des zugehörigen genRob®-genPCAN-Servers belegt werden,
  • ein Kartenserver genRob®-genMap muß im Netz sichtbar sein und
  • es müssen Routen (vgl. Kartenelemente) in genRob®-genMap vorhanden sein.

Von genRob®-genPCAN kommen letztlich die Meßwerte.  In genRob®-genMap sind die Routen abgelegt.

Sitzungen - Meßwerterfassung

Sind obige Voraussetzungen erfüllt, so kann man Meßwerterfassungen durchführen.  Dazu muß man zunächst über das Sitzungsmenü eine neue Sitzung anlegen.  Dann zeigen sich an den Enden einer jeden Route Bedienelemente.  Diese Bedienelement ermöglichen den Start, den Abbruch und das Stoppen von Meßerfassungen.  Später oder auch während einer Meßfahrt kann man über die Temperaturskala die Meßwertdarstellung einstellen.

Sitzungen unterbrechen

 Über das Sitzungsmenü werden Sitzungen unterbrochen, fortgesetzt und beendet.  Ist keine Sitzung aktiv, so kann man Aufzeichnungen von Sitzungen einsehen.  Bei der Anzeige von Sitzungen können verschiedene Anzeigeoptionen genutzt werden.

Änderung einer Route

 Unterbricht man eine Sitzung und ändert dann die Koordinaten einer Route, so werden für diese Route bereits erfaßte Daten gelöscht.
