Einrichtung

Die Kombination genRob®-genControl/-genMap ist bis auf eine möglicherweise notwendige Parametrisierung sofort einsatzbereit.  Der Kartenserver ist dann zunächst leer, aber bereit, Daten zu verwalten.  Zweckmäßig ist die Nutzung und gegebenenfalls Anpassung der mitgegebenen Start-Dateien, anstatt bei jedem Start alle Parameter einzugeben.

Beim ersten Lauf wird im Arbeitsverzeichnis eine Kartendatei mit Namen genRob.genMap.serialized (vgl. Persistenz) zur Speicherung von Kartendaten erzeugt.  Die erzeugte Karte ist zunächst leer.

Kartendateien, die von älteren Versionen von genRob®-genMap erzeugt worden, lassen sich u.U. nicht von dieser Version verwenden und werden einfach überschrieben.  Bewahren Sie sich deshalb eine Kopie dieser Dateien separat auf.  Wenden Sie sich an genMap@genRob.com, um Hinweise bzw. Werkzeuge für eine Datenübernahme auf diese Version von genRob®-genMap zu bekommen.
