Werkzeug

 Mit dem Paket genRob®-genPath wird zusätzlich noch ein Werkzeug ausgeliefert.  Es unterstützt die Einrichtung und Wartung von genRob®-genPath.

  • Ausführung
  • Parameter

Ausführung

 Der Aufruf erfolgt in folgender Weise: 
java  -jar genRob.genPath.tool.jar  [Parameter]

Parameter

 Die Parameterangabe folgt folgender Regel (in eine Kommandozeile schreiben): 
java  -jar genRob.genPath.tool.jar
		host:port Einheit Befehl
host:port gibt den Roblet®-Server mit genRob®-genPath-Modul an.  Kombinationen von Einheit und Befehl können aus der weiter unten stehenden Tabelle entnommen werden.

Erfolgt der Aufruf ohne Parameter, so wird angezeigt, was mit dem Werkzeug alles gemacht werden kann.

Nachfolgend die Einheit-Befehl-Kombinationen und ihre Funktion:

Einheit Befehl Funktion
Version get Hole Versionsinformationen zum genPath-Modul vom angegebenen Server.
Planner test Plane eine Bahn von (in Meter) -10:-10 nach 10:10 bei einem Fahrzeugradius von 0,5 m und gebe sie aus.
Planner plan Weitere Parameter (in Millimeter):  Radius AX AY BX BY

Plane eine Bahn von AX:AY nach BX:BY bei einem Fahrzeugradius von Radius und gebe sie aus.
